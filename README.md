# SmartMed - Medical Supply Chain Management System

Developed a full-stack MERN application for medical inventory and supply chain management using React.js, Node.js, Express.js, and MongoDB.

Designed RESTful APIs for inventory tracking, order processing, and user management.

Implemented role-based access control and integrated MongoDB for real-time data storage and retrieval.

## Technologies Used
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- User Authentication
- Inventory Management
- Order Processing
- Role-Based Access Control
- Delivery Tracking
